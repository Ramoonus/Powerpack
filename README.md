# Powerpack
Ramons Powerpack for WordPress

## Features ##

* EON.js
* FontAwesome
* D3.js
* Selectivizr
* Respond.js
* Lettering.js
* html5shiv
* Comic Sans everything
* Amplify.js
* Explorer Canvas (excancas)
* Enhance.js
* Microformat shiv
* MatchMedia.js
* Pointer Events polyfill (PEP)
* Globalize
* FitText
* Has.js
* IE9 mode
* eBay Shortcode
